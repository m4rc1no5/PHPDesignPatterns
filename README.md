<a name="installation"></a>
Installation
------------

After download PHPDesignPatterns You must use composer to load all dependecies.

```
    composer install -o
```

Update dependecies: 

```
    composer update -o
```

<a name="run"></a>
Run examples
------------

You can run examples using commands like this:

```
    php examples/TheObserverPattern/example_01.php
```
